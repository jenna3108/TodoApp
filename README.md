# TodoApp

Functionalities
1) Add task item
2) Remove task item
3) Complete task item

## Add task item
- Insert your task in the textbbox
- Press 'Add New Task' button to save item

## Remove task item
- Press the trash icon on the item that you want to remove

## Complete task item
- Press on the item row that you've completed
